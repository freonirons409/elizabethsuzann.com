How To...

  • Create image slider in content areas on pages and blog posts
    To create a slideshow out of a group of images on one of your pages or blog posts you simple need to upload the images to the content editor, and then make sure all of the images you want in the slideshow are wrapped in <p></p> tags. To do so, click the little < > icon in the top right of the content editor to open the code view, and then make sure your images look something like this:
    <p> <img src="http://placehold.it/200x200" alt="image alt text"/> <img src="http://placehold.it/200x200" alt="image alt text"/> <img src="http://placehold.it/200x200" alt="image alt text"/> <img src="http://placehold.it/200x200" alt="image alt text"/> <img src="http://placehold.it/200x200" alt="image alt text"/> </p>
    Note: make sure your images do not have any other tags wrapped around them and do not contain and style="" attributes. These can be added by Shopify without you noticing and can cause adverse effects.
  • Add hover captions to images
    Inside the ALT text of any image, you can place the following code to generate a hovering caption that overlays on your image, complete with code and button links:

    Basic syntax
    HOVER_This is a caption that will show up when hovering

    Alternate Syntax
    HOVER2_This is a caption that will show up when hovering and is a darkened version.

    Accepted HTML tags
    HTML tags can be used inside your caption but using a slightly different syntax. See below for equivalents:

    • Heading <h1>: [h1]Heading[/h1]
    • Heading <h2>: [h2]Heading[/h2]
    • Heading <h3>: [h3]Heading[/h3]
    • Heading <h4>: [h4]Heading[/h4]
    • Heading <h5>: [h5]Heading[/h5]
    • Heading <h6>: [h6]Heading[/h6]
    • Paragraph <p>: [p]paragraph text[/p]
    • Line Break <br>: [br]
    • Button <button>: [button]button text[/button]
    • Link <a>: [a href=|http://www.yourlink.com|]this is a link[/a]
  • Create tab blocks in product descriptions

    The product description will be default contain everything in one tab called "Description" without any further markup. But if you want to make new tabs, simply create a section using <h5> as the defintion for the start of a new tab, and whatever is wrapped in the <h5> will become the accordion header.

    Example:

    <p>This content will be in tab 1 and the toggle header will default to "Desription". Lorem ipsum dolor sit amet, consectetur adipisicing elit. Recusandae debitis corrupti nesciunt, sed itaque placeat possimus, adipisci delectus temporibus ullam!</p> <h5>Second Accordion Header</h5> <p>This will be accordion 2 content and the header will be "Second Accordion Header"</p>
  • Add slideshow to product page
    To create a slideshow that shows at the bottom of your product page, upload product images like you normally would and use the image editor to edit the ALT text of the image. In the ALT text use the prefix SLIDESHOW_ and the page will automatically know where to put your image. You can also use this in addition with the HOVER_feature, but make sure you put the HOVER_ after the SLIDESHOW_
  • Add diversity campaign block to product page
    To set an image as the diversity campaign block image (this cannot be a slideshow) use the prefix DIVERSITY_ in the ALT text of the product image you want to use. To make it even easier, use the People Generator to your left under the "Code Generators" tab to auto generate the code for you that is needed to display the details for each person featured in the diversity image. This generator will also include the DIVERSITY_ prefix so all you will need to do is copy and paste.
  • Add fabric types to products
    To make a product feature the fabric it is made of on the product and collections pages, simple go to your products edit page and under tags add the fabric or fabrics you want it to be categorized under. Accept fabrics are currently:
    • Silk
    • Cotton
    • Hemp
    • Linen
    • Wool
    • Leather

    If more types need to be added, contact your developer or you can do it yourself by editing the tag-looper.liquid snippet and you will see the list of types there.

  • Make images open in lightbox

    Any image you want to open in a lightbox simply add "lightbox" to the alt text of that image.
